Loading animation — Northlake University

Overrides .k-loading-mask .k-loading-image (56×56) with concentric lake ripples pulsing from a gold spark. ~1.2 s seamless loop, legible on the frosted mask. A Lottie variant powers a richer full-page splash.

Integration

The in-app spinner is pure CSS — no runtime. Drop the LESS below in to supersede loading-animation.less; it targets the real Kendo selectors. The Lottie splash reuses the same player as the login entrance.

▸1 · Drop-in LESS supersedes loading-animation.less 
// Replaces the generic Kendo ring with lake ripples + a gold spark.
.k-loading-mask {
  display:flex; flex-direction:column; align-items:center; justify-content:center; gap:14px;
  background: rgba(250,248,242,.60);                // frosted paper
  backdrop-filter: blur(3px);
  .k-loading-color { display:none; }

  .k-loading-image {
    position:relative; margin:0; width:56px; height:56px;
    background: url("data:image/svg+xml,…gold north-star…") center/22px no-repeat;
    animation: nl-spark 1.2s ease-in-out infinite;  // gentle breathe
  }
  .k-loading-image::before,
  .k-loading-image::after {
    content:""; position:absolute; left:50%; top:50%;
    width:20px; height:20px; margin:-10px 0 0 -10px; border-radius:50%;
    border:2px solid var(--color-accent);
    opacity:0; animation: nl-ripple 1.2s cubic-bezier(.22,1,.36,1) infinite;
  }
  .k-loading-image::after { border-color:var(--color-tertiary); animation-delay:.6s; }

  .k-loading-text { /* keep the hook — now a legible caption */
    position:static; text-indent:0; font-size:var(--font-size-sm);
    font-weight:500; color:var(--color-primary);
  }
}
@keyframes nl-ripple { 0%{transform:scale(.45);opacity:0} 18%{opacity:.75} 100%{transform:scale(2.7);opacity:0} }
@keyframes nl-spark  { 0%,100%{background-size:22px} 50%{background-size:25px} }

@media (prefers-reduced-motion: reduce){      // hold a calm, static mark
  .k-loading-mask .k-loading-image,
  .k-loading-mask .k-loading-image::before,
  .k-loading-mask .k-loading-image::after { animation:none; }
}
▸2 · Lottie splash variant full-page boot 
// Richer full-page splash — same MIT player as the login entrance.
<div class="nl-splash"><div class="nl-splash__mark"></div></div>

const reduce = matchMedia('(prefers-reduced-motion: reduce)').matches;
fetch('/branding/northlake-loading.json')
  .then(r => r.json())
  .then(data => {
    const anim = lottie.loadAnimation({
      container: document.querySelector('.nl-splash__mark'),
      renderer: 'svg', loop: !reduce, autoplay: !reduce, animationData: data
    });
    if (reduce) anim.goToAndStop(8, true);   // static frame
  });
★
Per-customer reskin. The ripples read --color-accent / --color-tertiary; the caption reads --color-primary. Change those in northlake-theme-vars.css (and the spark's fill / northlake-loading.json) — no animation code changes.
